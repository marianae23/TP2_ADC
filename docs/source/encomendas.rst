encomendas.py
=============

Descrição
=========

Módulo responsável pela gestão das encomendas.

Funcionalidades
===============

Gestão de Encomendas
====================

- listar encomendas
- procurar encomenda por ID
- criar encomendas
- cancelar encomendas

Validações
==========

- validar stock
- validar datas disponíveis
- validar limite mensal
- validar métodos de envio
- validar métodos de pagamento

Cálculos
========

- calcular preço com promoção
- calcular total da encomenda

Alterações
==========

- alterar data de entrega

Consultas
=========

- listar encomendas por estado
- listar encomendas pendentes
- listar encomendas entregues
- listar encomendas canceladas

Relatórios
==========

- total de vendas
- ordenar encomendas por data
- resumo de encomenda

Tempo e Alertas
===============

- estimar tempo de espera
- alertas de atraso
