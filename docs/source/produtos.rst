produtos.py
===========

Descrição
=========

Módulo responsável pela gestão do catálogo de produtos.

Funcionalidades
===============

Listagem
--------

- listar produtos
- listar produtos disponíveis
- listar produtos novos
- listar produtos em promoção

Pesquisa
========

- procurar produto por ID
- procurar produto por nome

Filtros
=======

- filtrar por categoria
- filtrar por sabor
- filtrar por tamanho
- filtrar por cor
- filtrar por preço
- filtrar por intolerâncias

Promoções
=========

- aplicar promoções
- remover promoções
- calcular preço final

Avaliações
==========

- avaliar produto
- listar avaliações
- calcular média de avaliações
- identificar produto mais bem avaliado

Recomendação
============

- recomendar produto
- produto do dia

Resumo
======

- gerar resumo detalhado do produto
