dados.py
========

Descrição
=========

Módulo responsável pelo armazenamento central de dados do sistema em memória.

Estruturas de Dados
===================

Produtos
--------

Contém:

- identificador
- nome
- categoria
- preço
- stock
- disponibilidade
- sabores
- tamanhos
- cores
- intolerâncias
- promoções
- estado de novidade
- avaliações

Encomendas
----------

Contém:

- identificação do cliente
- produto associado
- quantidade
- personalização
- método de envio
- método de pagamento
- data desejada
- estado
- comentários
